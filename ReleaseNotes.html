März 2023

Alle Neuerungen mit dem Release 2.14.0

Optimierte Kategorie Navigation:

Das Kategorien-Feld bleibt während der Navigation innerhalb einer Artikelauflistung bestehen. Es werden Kategorien, Hauptwarengruppen und Unterwarengruppen aufgeführt, in denen sich Artikel aus der Suche befinden. Diese lassen sich über den Pfeil ein- oder ausklappen. Die aktuell ausgewählte Warengruppe wird unterstrichen. Mit einem Klick auf eine Warengruppe oder Kategorie gelangt man direkt zu den entsprechenden Suchergebnissen. Dadurch ist eine Aufwärts- und Abwärtsnavigation möglich.

Bearbeiten von Listen:

In den Einkauflisten kann nun in einen „Bearbeiten“-Modus gewechselt werden. In diesem Modus können Standardmengen verändert werden, Listenbeschreibung und -titel angepasst oder die Liste gelöscht werden. Der Ordersatz und die Etikettenliste können nicht gelöscht werden. Verfügt ein Benutzer über Disponentenrechte, so können im Bearbeitungs-Modus lediglich die Standardmengen angepasst werden. Dadurch wird eine Fehlbedienung verhindert und die Oberfläche wirkt deutlich aufgeräumter.

Pop Up, sobald Standardmengen in den Warenkorb gelegt wurden:

Wurden Artikel mit ihrer Standardmenge in den Warenkorb gelegt, so wird ein Pop Up ausgespielt. Auch wenn versucht wird, die Standardmengen ein zweites Mal in den Warenkorb zu legen, erscheint ein Pop Up, welches darauf hinweist, dass sich die Artikel mit ihrer Standardmenge bereits im Warenkorb befinden.

Optimierungen

- Darstellung von Inhalten in der mobilen Version: Startseitenkacheln werden in der mobilen Version so angezeigt, dass der gesamte Text lesbar ist.

- Verbessertes Userfeedback bei unvollständigem Registrierungsformular: Wenn Pflichtfelder beim Abschicken der Registrierung nicht ausgefüllt sind, wird eine Fehlermeldung angezeigt und das entsprechende Feld wird rot umrahmt.

- Befinden sich im aktuellen Warenkorb Bestellartikel ohne Vorlaufzeit, so wird für diese Artikel kein separates Lieferdatum angezeigt.

- Die Übersichtlichkeit der Aktionen wurde durch Schattierungen und einen größeren Abstand zwischen den Aktionen verbessert.

Fehlerbehebungen:

- Die Darstellung von Mindestmengen wurde korrigiert.

- Die Berichte werden auch dann geladen, wenn sie von einem Zentraleinkäufer aufgerufen werden.

- Die Bestelldetails lassen sich wieder drucken.

Januar 2023

Alle Neuerungen mit dem Release 2.13.0

Uhrzeitabhängige Bestellartikel:

Die Logik von Bestellartikeln mit Bestellvorlauf wurde mit dem Handling für uhrzeitabhängige Bestellartikel verknüpft. Wird ein uhrzeitabhängiger Bestellartikel nach der Wechseluhrzeit bestellt und ist für diesen Artikel eine Wiederbeschaffungszeit hinterlegt, so wird der Artikel in einen neuen Teilwarenkorb gelegt. Wird ein uhrzeitabhängiger Bestellartikel mit Wiederbeschaffungszeit vor der Wechseluhrzeit bestellt, so verhält sich der Artikel im Warenkorb wie ein normaler Lagerartikel.

Bei Unklarheiten kontaktieren Sie uns bitte direkt.

Optimierungen:

- An Ordersatz-Artikeln wird nun ein Ordersatz-Icon ausgespielt.

- Das Wiegeartikel-Icon wird nicht mehr ausgespielt.

- Das Direkt-Frisch-Icon wurde aktualisiert

- In den persönlichen Informationen wird einheitlich der Begriff „Passwort“ und nicht „Kennwort“ verwendet.

Fehlerbehebungen:

- Icons von der Artikeldetailseite werden wieder auf dem Artikelpass angedruckt.

- Im Warenkorb wurde eine Fehlermeldung für direkt.frisch Artikel ausgespielt, obwohl die Mindestbestellmenge erfüllt war. Diese Fehlermeldung wird nun nicht mehr ausgespielt.

September 2022

Alle Neuerungen mit dem Release 2.11.0

Galerieansicht:

Im Shop kann nun zwischen einer Listenansicht und einer Galerieansicht gewählt werden (siehe gelb markierte Buttons im Screenshot). Die Galerieansicht eignet sich durch die großen Produktabbildungen insbesondere dafür, das Sortiment zu entdecken, während die Listenansicht genutzt werden kann, um schnell an Informationen zu gelangen und eine Bestellung aufzugeben. Die Galerieansicht ist im Gesamtkatalog und über das Burger-Menü nutzbar. Im Warenkorb und den persönlichen Einkaufslisten erscheinen die Artikel nur in der Listenansicht. Die Galerieansicht kann sowohl in der Desktop-Ansicht als auch in der mobilen Ansicht ausgewählt werden. Die ausgewählte Ansicht wird im Rahmen eines Logins gespeichert.

Optimierungen:

- Dummy-Artikelbilder werden jetzt auch unter iOS ausgespielt

- Um den Warenkorb übersichtlicher zu gestalten, wurde das Lieferdatum neben Lager-/Bestellartikel platziert.
